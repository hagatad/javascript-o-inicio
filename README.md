# Javascript - O início
> Início dos estudos sobre JavaScript

----

## Conteúdo inicial

- 1 - Variáveis e Tipos de Dados

    var, let, const
    Tipos: string, number, boolean, null, undefined, object, array

- 2 - Operadores e Expressões

    Aritméticos, comparação (==, ===), lógicos (&&, ||, !)
    Precedência e agrupamento

- 3 - Estruturas de Controle

    Condições: if, else, switch
    Loops: for, while, for...of

- 4 - Funções

    Declaração e chamada de funções
    Parâmetros, retorno
    Arrow functions (=>), escopo e closures

- 5 - Arrays e Objetos (Coleções de Dados)

    Criar, acessar, modificar
    Métodos úteis: map, filter, forEach, push, etc.
    Desestruturação ({} e [])

- 6 - Manipulação do DOM (HTML com JS)

    getElementById, querySelector
    Alterar texto, classes, estilos
    Criar/remover elementos

- 7 - Eventos e Interações com o Usuário

    addEventListener
    Eventos comuns: click, submit, change
    Manipulação dinâmica com base na ação do usuário

- 8 - Tratamento de Erros

    try, catch, finally
    Mensagens de erro e debug (console.log, console.error)

- 9 - Boas Práticas e Organização de Código

    Nomeclatura clara
    Código limpo e reutilizável
    Comentários úteis

- 10 - [MAIS IMPORTANTE] – Lógica de Programação

    Pensar em passos lógicos para resolver problemas
    Montar algoritmos simples (entrada → processamento → saída)
    Fundamental para entender qualquer linguagem, inclusive JS
